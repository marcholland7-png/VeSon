# Veson

A dark, futuristic personal assistant dashboard. Static HTML/CSS/JS, no build step.

**Stage 1:** dashboard shell, side nav, greeting, and a Home card that pulls today's events from the same sync-code mechanism as the [Calendar app](https://github.com/marcholland7-png/Calendar-app), via Settings.

## Run locally
Open `index.html` directly, or serve the folder (e.g. `python -m http.server`) and visit `http://localhost:8000`.

## Deploy
GitHub Pages → Settings → Pages → Deploy from branch → `main` / `root`. No build step required.
